Ο κώδικας υπολογίζει και εμφανίζει την ακολουθία aliquot (aliquot sequence) που ξεκινά από έναν αρχικό θετικό ακέραιο αριθμό.

Η aliquot sequence είναι μια ακολουθία που ξεκινά από έναν αριθμό n₀, και κάθε επόμενο στοιχείο είναι το άθροισμα των θετικών διαιρετών του προηγούμενου αριθμού (εκτός του ίδιου του αριθμού). Η ακολουθία σταματά όταν φτάσει στο 0, ή σε αριθμό που υπερβαίνει ένα όριο.LIMIT: μέγιστος επιτρεπόμενος αριθμός (10^15).

start: αριθμός από τον οποίο θα ξεκινήσει η ακολουθία.

max_len: μέγιστο πλήθος όρων που θα υπολογίσει το πρόγραμμα (ή 0 για απεριόριστο).

mode: καθορίζει αν ο χρήστης θέλει να δει ολόκληρη την ακολουθία (f) ή μόνο το μήκος της (l).Αυτή η συνάρτηση υπολογίζει το άθροισμα των θετικών διαιρετών του n, εκτός του ίδιου του αριθμού.

Αν n = 1, τότε επιστρέφει 0, γιατί ο 1 δεν έχει σωστούς διαιρέτες εκτός από τον εαυτό του.

Ξεκινά το άθροισμα με 1, γιατί το 1 είναι διαιρέτης κάθε αριθμού. Παράδειγμα:
Για n = 12, οι θετικοί διαιρέτες του (εκτός του ίδιου) είναι 1, 2, 3, 4, 6.
div_sum(12) = 16. Ο χρήστης επιλέγει:

'f' → εμφάνιση ολόκληρης της ακολουθίας.

'l' → εμφάνιση μόνο του μήκους.
